# OmniCampus - Integrated Campus Management System

OmniCampus is a comprehensive campus management system for students and administrators, featuring Library Management, Lost & Found, Student Clubs, and an AI-powered assistant.

## Features

### For Students
- **Library Management**: Browse books, check availability, request books, view loans and fines
- **Lost & Found**: Report lost items, view found items, check potential matches
- **Student Clubs**: Join clubs, view events, pay dues, connect with peers
- **AI Assistant**: Get recommendations and answers to campus-related questions
- **Personal Dashboard**: View active loans, upcoming events, book recommendations

### For Administrators
- **Library Management**: Add/edit books, manage loans, approve book requests
- **Lost & Found**: Moderate reports, manage matches, update item statuses
- **Club Management**: Create clubs, approve memberships, manage events
- **User Management**: View registered students, manage accounts
- **Analytics Dashboard**: View system statistics and reports

## Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: Python Flask with REST API
- **Database**: SQLite (development) / PostgreSQL (production)
- **Authentication**: JWT-based authentication
- **AI Features**: Python-based recommendation engine and chatbot
